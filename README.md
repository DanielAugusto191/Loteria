# Conteudo

* Estatistica com grafico MatPlotLib.
* VirtualEnv.
* Tkinter Python 3.

<a name="Install"></a>
# Uso e instalação
Instalação:
Baixe os arquivos, abra o terminal:

1. sudo apt install python3-pip
2. sudo pip3 install -r requirements.txt
3. sudo apt-get install python3-tk

Uso:
No terminal:
1. source env/bin/activate
2. pytohn3 main.py
